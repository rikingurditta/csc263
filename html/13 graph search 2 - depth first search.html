Depth-First Search

  • BFS: first discovered -> first explored

    • implemented with a queue

  • DFS: last discovered -> first explored

    • implemented with a stack

Graph Search Recap

Algorithm keeps track of:

  • colour[v]: white, grey, or black:

    • white: undiscovered
    • grey: discovered but not completely explored
    • black: explored

  • p[v]: iff was discovered while exploring

  • d[v]: time when was discovered (different from d[v] in BFS!)

  • f[v]: time when exploration of was finished
